Signature Verification\n\nEach provider uses HMAC-SHA256 for webhook verification:\n\n**Stripe:**\n```typescript\nconst signedContent = `${timestamp}.${rawBody.toString()}`;\nconst signature = crypto\n  .createHmac('sha256', secret)\n  .update(signedContent)\n  .digest('hex');\n```\n\n**PayPal:**\n```typescript\nconst verificationString = [\n  transmissionId,\n  transmissionTime,\n  webhookId,\n  sha256(payload)\n].join('|');\n```\n\n**Chapa:**\n```typescript\nconst signature = crypto\n  .createHmac('sha256', secret)\n  .update(JSON.stringify(payload))\n  .digest('hex');\n```\n\n### 2. Idempotent Processing\n\nWebhooks are deduplicated using idempotency keys:\n\n```typescript\n// Check if already processed\nconst existing = await prisma.webhookLog.findUnique({\n  where: { idempotencyKey },\n});\n\nif (existing && existing.processedAt) {\n  return existing.id; // Already processed\n}\n```\n\n### 3. Exponential Backoff Retry\n\nFailed webhooks are retried with exponential backoff:\n\n```\nAttempt 1: 1s delay\nAttempt 2: 2s delay\nAttempt 3: 4s delay\nAttempt 4: 8s delay\nAttempt 5: 16s delay (max: 5 minutes)\n```\n\n### 4. GDPR Compliance\n\n**Data Minimization:**\n- Only collect necessary fields\n- Mask sensitive data in logs\n- Validate PII fields\n\n**Consent Management:**\n- Record user consent\n- Allow consent revocation\n- Track consent versions\n\n**Data Retention:**\n- Automatic deletion after 90 days (configurable)\n- Anonymize deleted records\n- Maintain audit trail\n\n**Subject Access Requests:**\n```typescript\nconst data = await gdprService.getDataSubjectAccess(userId);\n// Returns all personal data for this user\n```\n\n**Right to be Forgotten:**\n```typescript\nawait gdprService.deleteUserData(userId);\n// Anonymizes all personal data\n```\n\n### 5. Multi-Currency Support\n\n```typescript\n// Convert payment amount to user's currency\nconst convertedAmount = await convertCurrency(\n  100,      // amount\n  'USD',    // from currency\n  'ETB'     // to currency (Ethiopian Birr)\n);\n// Result: 13000 ETB (using current exchange rate)\n```\n\n### 6. Internationalization (i18n)\n\n```typescript\n// Translate message to user's locale\nconst message = i18nService.translate(\n  'payment.success',\n  'am',  // Amharic\n  { amount: '500', currency: 'ETB' }\n);\n// Result: \"бҠӯбҚҚбӢ« бҲөбҠ¬бүібҲӣ бҠҗбү бҲӯ! бүҘбҲӯ 500 бү°бҢЁбҲқбҲ® бҠҗбү бҲӯбҚў\"\n\n// Format currency for display\nconst formatted = i18nService.formatCurrency(\n  500,\n  'ETB',\n  'am-ET'\n);\n// Result: \"Br 500.00\"\n```\n\n---\n\n## Event Processing Flow\n\n### Payment Success Flow\n\n```\n1.
